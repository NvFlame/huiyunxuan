# 隐私说明

绘云轩当前版本不提供云同步服务，主要数据保存在用户设备本地。

## 本地保存的数据

- 诗词库、诗词内容、译文、注释、赏析、学习笔记。
- 收藏关系、学习进度、训练进度、默诵值。
- API 配置，包括用户手动填写的 API Key、Base URL、模型名称和搜索接口配置。
- AI 助手对话记录。

## 备份文件

App 的备份包用于迁移或恢复本地数据，可能包含 API Key、学习笔记、AI 对话记录和完整诗词库。备份包应视为个人敏感文件，不应上传到公开仓库、网盘公开链接或 issue 附件。

## 网络请求

只有在用户配置并使用 AI 助手、模型列表、API 测试、联网搜索等功能时，App 才会向用户配置的服务商发起请求。请求内容可能包含用户输入、当前诗词内容或待补全的诗词资料。

## 开源仓库注意事项

请勿提交以下文件：

- API Key 或 `.env` 文件。
- App 备份包。
- Android 签名密钥、`key.properties`、`local.properties`。
- 包含个人诗词库或聊天记录的数据库文件。

如果你 fork 或二次开发本项目，请自行审查第三方服务商的隐私政策与数据处理方式。
