# 绘云轩

绘云轩是一款 Flutter 安卓诗词学习与训练 App，目标是把本地诗词库、学文、展才训练、收藏、备份、AI 助手与格律辅助整合在一个轻量应用里。
本项目借助韵书与大模型实现了全自动的诗词格律校对，并可以校对诗词是否押韵与词的叠句，给出格律错处标注。目前支持26个常见词牌校对。
用户可以借助智能体自动联网搜索添加诗词，并在诗词库中进行编辑，或让智能体进行补充。
项目目前以 Android 端为主，所有诗词库与学习数据默认保存在本机本地数据库中，不依赖云同步。

## 制作动机

截至此时，市面上缺少能够在背诵默写上友好交互的诗词APP，也没有将智能体应用到诗词学习中的APP，亦缺少能彻底解决因多音字造成的无法判断格律正误的问题。我希望能将在学习的过程中直接调用模型、添加诗词、并进行背诵验证，于是开发了该APP。该项目可以有效进行古诗词训练，无论是诗词爱好者还是学生，均适宜使用。因时间仓促，目前仅内置了唐诗三百首一个诗词库。

## 当前功能

- 诗词库管理：新建、编辑、删除诗词库，浏览库内诗词。
- 诗词元素：标题、作者、朝代、序、正文、译文、注释、学习笔记、赏析、备注。
- 收藏夹：内置收藏库，可将诗词收藏到收藏夹。
- 学习模式（学文）：阅读诗词、查看注释/译文/赏析/学习笔记。
- 默写模式（展才）：秀才、举人、贡生、进士四种训练模式，对应不同难度。批改模式分为即时批改与最终批改。
- 默诵值：记录首次通过进士模式的诗词。
- 格律辅助：近体诗基础审查，部分词牌词谱审查，支持人工/智能校准平仄与韵部。
- AI 助手：可配置 OpenAI 兼容 API 与搜索接口，用于问答、补全、添加或更正诗词资料。目前搜索接口支持Tavily以及博查。
- 导入导出：支持诗词/诗词库导入导出，以及整包备份与恢复。

## 构建 APK

调试版：

```powershell
flutter build apk --debug
```

发布版：

```powershell
flutter build apk --release
```

发布 APK 通常位于：

```text
build\app\outputs\flutter-apk\app-release.apk
```

## 数据保留

同一个包名下覆盖安装新版 APK，Android 会保留 App 本地数据；卸载后重新安装会清空本地数据。升级前建议先在 App 设置中导出备份。
